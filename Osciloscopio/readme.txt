1. Contents
-----------
Files contained in this package include:

 1  ) Libs.c              -- The main program
 2  ) registros.h         -- includes and genral defines
 3  ) systickControl      -- Functions for initialization and uses of systick
 4  ) timer               -- Functions for initialization and uses of timers
 5  ) adc                 -- Functions for initialization and uses of adcs
 6  ) serial              -- Functions for initialization and uses of serial port
 7  ) gpioControl         -- Functions for initialization and lower level operation
 8  ) oscilador           -- PLL configuration
 9  ) teclado             -- Functions for key command analysis and execution
 10 ) app                 -- Basic type definitions and general functions
 11 ) display LCD         -- Functions for screen related operations
 12 ) readme.txt          -- This file.
